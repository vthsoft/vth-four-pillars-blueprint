# VTH · Four Pillars Blueprint

Marco de arquitectura y entrega basado en cuatro pilares integrados:

1. Software a medida robusto  
2. Cloud + FinOps eficiente  
3. Seguridad integral desde el diseño  
4. IA como capacidad core  

Este repo reúne notas de arquitectura, diagramas y ejemplos para aplicar los cuatro pilares en proyectos reales.

## Contenido (MVP)

- `docs/01-overview.md` – Visión general del marco y principios.  
- `docs/02-software.md` – Clean Architecture, DDD, testing, observabilidad.  
- `docs/03-cloud-finops.md` – Patrones cloud, métricas y optimización de costos.  
- `docs/04-security.md` – Zero-Trust, IAM, cifrado y compliance.  
- `docs/05-ai.md` – Casos de uso de IA, RAG, asistentes internos, automatización.

> La idea es compartir cómo pensamos la arquitectura, no exponer código de clientes.
